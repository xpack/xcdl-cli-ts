# The `minimumXcdlRequired` definition

:::note

Work in progress.

:::

## Purpose

Identify the minimum required **xcdl** version.

## Syntax

```json {2}
{
    "minimumXcdlRequired": "<semver>",
}
```

The value of the `minimumXcdlRequired` property is a string with a version in [SemVer](https://semver.org) format.

## Default value

If missing, no checks are performed on the **xcdl** version.

<MinimumRequired name="xcdl" fileName="xcdl" />

```json {2}
{
    "minimumXcdlRequired": "2.0.0",
    ...
}
```
